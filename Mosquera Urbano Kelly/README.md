## **Descripción**

Este repositorio contiene el desarrollo del Taller No. 3 correspondiente a la asignatura Inteligencia Artificial, cuyo propósito es aplicar técnicas de estadística descriptiva para analizar la variable Apps Descargadas a partir de un conjunto de datos suministrado por el docente.

Durante el desarrollo del taller se realizaron los cálculos estadísticos necesarios para construir una tabla de distribución de frecuencias y diferentes representaciones gráficas que permiten interpretar el comportamiento de los datos.



**Objetivo General**

Aplicar métodos de estadística descriptiva para analizar la variable Apps Descargadas, construyendo tablas de frecuencias y representaciones gráficas que permitan interpretar el comportamiento de los datos y apoyar la toma de decisiones.

**Objetivos Específicos**

* Identificar las características generales del conjunto de datos.
* Determinar el número de observaciones de la muestra.
* Calcular valor mínimo, valor máximo y rango.
* Aplicar la Regla de Sturges para determinar el número de intervalos.
* Calcular la amplitud de clase.
* Construir la tabla de distribución de frecuencias.
* Calcular la marca de clase.
* Determinar las frecuencias absoluta, relativa, acumulada y relativa acumulada.
* Elaborar e interpretar el histograma.
* Elaborar e interpretar el polígono de frecuencias.
* Elaborar e interpretar la ojiva o curva acumulada.
* Analizar estadísticamente los resultados obtenidos.


## **Desarrollo del Taller** 

Durante la realización de esta actividad se llevaron a cabo las siguientes etapas:

1. **Análisis de la variable**
Población.
Muestra.
Unidad de análisis.
Tipo de variable.

3. **Cálculos estadísticos**
   
Número total de observaciones.
Valor mínimo.
Valor máximo.
Rango.
Número de intervalos.
Amplitud de clase.

4. **Tabla de distribución de frecuencias**
Se construyó una tabla estadística que incluye:

Intervalos de clase.
Marca de clase.
Frecuencia absoluta (fi).
Frecuencia relativa (hi).
Frecuencia acumulada (Fi).
Frecuencia relativa acumulada (Hi).
4. Representación gráfica

5.**Para facilitar la interpretación de los datos se elaboraron los siguientes gráficos:**

Histograma de frecuencias.
Polígono de frecuencias.
Ojiva o curva acumulada.

Cada representación gráfica incluye su respectivo análisis técnico.
